## 开发

请确保本地安装了 Node.js 和 pnpm 以及开发目标浏览器， 同时建议你用 Visual Studio Code 进行开发。

```sh
# 安装依赖
pnpm install

# 运行项目
pnpm dev
```

运行项目会自动打开 chrome 浏览器并添加扩展至扩展列表。
