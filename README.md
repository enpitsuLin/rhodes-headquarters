## 开发

```sh
git clone git@github.com:enpitsuLin/rhodes-headquarters.git
cd rhodes-headquarters
pnpm i
pnpm dev
```

## 调试

### firefox:

```bash
pnpm start:firefox
```

### chromium:

```bash
pnpm start:chromium
```

## 构建

```bash
pnpm build
```
